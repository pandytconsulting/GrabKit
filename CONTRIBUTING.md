# Contributing to GrabKit

Thanks for helping improve GrabKit.

## Development

1. Install a Flutter version compatible with `pubspec.yaml`.
2. Run `flutter pub get`.
3. Run `dart format --set-exit-if-changed .`.
4. Run `dart analyze`.
5. Run `flutter test`.

Keep changes focused and include tests for new behavior. Never add real API
endpoints, access tokens, customer data, or other private application details
to fixtures, examples, or documentation.

## Pull Requests

- Describe the problem and the behavior change.
- Include screenshots for visible UI changes.
- Add a changelog entry for user-facing changes.
- Confirm that formatting, analysis, and tests pass.
